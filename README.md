How to Run the Project



Navigate to the Server Directory: Open your terminal and navigate to the server directory of the project.

cd path/to/your/project/server
Launch the Node.jsServer: Start the server by running the following command:


node server.js


Visit the Application: Open your web browser and go to http://localhost:3000.

Test on Multiple Browsers: To try out the code, open the application on at least two different browsers.

Future Hosting Plans: Please note that this project will be hosted on a domain in the future. Stay tuned for updates!
